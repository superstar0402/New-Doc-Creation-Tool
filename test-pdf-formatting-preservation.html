Function: PDF Export in handleExport
Logic: Uses convertFormattedContentToHTML() to preserve formatting
Code Implementation: 
// Always use formatted content if available to preserve bold/italic/underline in PDF
const contentHTML = (block.formattedContent && block.formattedContent.length > 0)
  ? convertFormattedContentToHTML(block.formattedContent)  // ✅ Preserves formatting
  : block.content.replace(/\n/g, '
');                 // ✅ Fallback for plain text
